# Hussain Haider | Portfolio

A personal portfolio website built with React and Vite, showcasing my work as a Software Engineering student at ITU Lahore.

## 🔗 Pages

- **Home** — Landing page introduction
- **About** — Background, focus areas, and what I do
- **Skills** — Technologies and tools I work with
- **Projects** — Selected project showcase
- **Contact** — Get in touch

## 🛠️ Built With

- [React](https://react.dev/) 19
- [Vite](https://vitejs.dev/) — build tool & dev server
- [React Router](https://reactrouter.com/) — client-side routing
- ESLint — code linting

## 🚀 Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/HussainHaider07/portfolio_app.git
cd portfolio_app
npm install
```

Run the development server:

```bash
npm run dev
```

This starts a local server (usually at `http://localhost:5173`) — open it in your browser to view the site.

### Other commands

```bash
npm run build     # Create a production build
npm run preview   # Preview the production build locally
npm run lint       # Run ESLint checks
```

## 📁 Project Structure

```
src/
├── components/   # Reusable UI components (Navbar, Footer, etc.)
├── pages/        # Route-level pages (Home, About, Skills, Projects, Contact)
├── data/         # Static content data
├── styles/       # Page and component styles
```
